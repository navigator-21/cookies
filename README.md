# cookies
This object is for working with cookies in JS environments.

Обект cookies обладает тримя методами:

	1. set() - принимает обект из 8 опций {} :

        ---------------------------------------------------------------------------------------- 

		1. name : "name",   -   Название cookie (обязательный параметер)

        ---------------------------------------------------------------------------------------- 

        2. value : "value",    -   Значение cookie (по умолчанию пуста строка)

        ---------------------------------------------------------------------------------------- 

        3. maxage : 2592000 ,   -   В ремя жизни cookie, выставляется в секундах < ((60*60)*24)*30 = 30 дней >, альтернатива expires, определяет срок действия куки в секундах с текущего момента. 

        ---------------------------------------------------------------------------------------- 

        4. expires : {
            year:       "xxxx", // год
            month:      "xx",   // месяц
            date:       "xx",   // день 
            hours:      "xx",   // часы
            minutes:    "xx",   // минуты
            seconds:    "xx",   // секунды
          }, 
          -   Это обект из набора опций которые позволяют выставить точную дату и время удаления cookie (необязательный параметер)

        ---------------------------------------------------------------------------------------- 

        5. domain : "site.com"   -   опция domain позволяет нам разрешить доступ к куки для поддоменов.

        ----------------------------------------------------------------------------------------    

        6. path : "/mypath",    URL-префикс пути, куки будут доступны для страниц под этим путём. Если куки установлено с path:"/admin", то оно будет доступно на страницах "/admin" и "/admin/something", 
                                но не на страницах "/home" или "/adminpage". По умолчанию указан в качестве пути корень path:"/", чтобы наши куки были доступны на всех страницах сайта.
            
        ----------------------------------------------------------------------------------------    

        7. secure : true   -   Делает куки доступным только при использовании HTTPS. (по умолчанию false)

        ---------------------------------------------------------------------------------------- 

        8. samesite : "strict || lax", // запрещает браузеру отправлять куки с запросами, поступающими извне, помогает предотвратить XSRF-атаки.(по умолчанию отключена)

        ----------------------------------------------------------------------------------------
